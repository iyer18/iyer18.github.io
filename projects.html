SafeWalk

×

Android Tablet Application ~ Java using Android Studio & IntelliJ

Product Description: Safe Walk is a campus safety system (currently done manually) that allows people on campus to request that someone walk with them from one location to another on campus, this is especially useful for new students. The aim of this project is to replicate the Safewalk framework, accordingly it is done in 2 parts: server and client. WordSearch

×

Stack & Queues ~ C++

Product Description: This is a 3 part project.

  • Implemented a growable stack using a linked list format.
  • Used the created stack class to solve wordsearch problems such as below.
    ABCDE
    SGAIJ
    KTMCO
    PQRKT
    UVWXY
    Algorithm:
    • Search for the first letter in the word.
    • If found, push the location(coordinates) onto stack, and search the surrounding cells for the next letter
    • If the next letter is also found, push that location onto the stack
    • Repeat until all letters are found
    • If, at any point, the next letter is NOT found, pop the last location off the stack and resume the search from the previous stage.
    • If the word is found, print out the coodinates in the correct order
    • If not, print "error"
  • Implemented a double-ended queue using the array-doubling strategy and an array format.

Cuckoo Hashing

×

Hash & Heaps ~ C++

Product Description: This is a 2 part project.

  • Implemented the Cuckoo Hashing mechanism with the following functions: insert, remove, lookup and maxloop. Please refer here for a demonstration of the Cuckoo Hashing mechanism.

    • The following 2 hash functions are used in my project: h1(x) = xmod(r) & h2(x) = xmod(r-1)

    • Insert: adds a new element to the hash table, this may require rehashing or element swaping
    • Remove: returns removed number after removing it from the heap
    • Lookup: returns if number is found
    • Maxloop: returns if maxloop is reached

  • Implemented a binary heap useing a binary tree format. It can be used as max­heap or a min­heap and has the following functions: Peek, Insert, Extract, Create, Heapify, Size
    • Peek: finds max key or find min key
    • Insert: adds a new element to the heap
    • Extract: returns min or max key after removing it from the heap
    • Create: create an empty max or min heap.
    • Heapify: create the heap out of a given array of elements.
    • Size: return the number of items in the heap.
