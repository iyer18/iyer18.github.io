Nomad

IOS Mobile Application ~ Swift, XCODE

Product Description: There are so many people of all ages who love to go on road trips, but documenting such trips is not so easy. The goal of this app is to pin pictures at locations with a description in order to have a visual scrapbook at the end of the trip. Even though there currently are some travel blogging and GPS tracking applications that can help track your trips, there is no single application that provides a cohesive overview. We can address this problem by organizing the different locations, media, and comments in a chronological list view for each trip with an additional mapview with pinpoints at each entry.

Personal Contribution: This project was done for my Software Engineering class at Purdue. I was part of a 4 person team, and each of us had our own role to play in the project: 2 of us worked on back-end and 2 on front-end. My role focused on the back-end development of the application; more specifically with respect to the storing and loading of the inputed data, however as all of us made it a point to be aware of and understand each other's parts as well, I also know how the User Interface and User Experience has been set up.
